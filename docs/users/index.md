# Almena Network for Users

Almena Network is a decentralized identity platform that gives you complete control over your digital identity. Unlike traditional systems where a company owns your account, here **you own your identity**.

## What is Almena Network?

Almena Network is built on open standards (W3C) and consists of two main applications:

- **Wallet** — Your personal identity wallet where you create and manage your decentralized identity (DID).
- **Desktop** — An administration console for organizations that issue or request verifiable credentials.

Both applications connect to a background service called the **Daemon**, which handles peer-to-peer networking and communication between nodes.

## Available Features

### Wallet

The wallet is currently in its initial phase. You can:

- **Create a new account** — Set up your identity with a secure password.
- **Password validation** — Real-time feedback ensures your password meets security requirements (minimum 8 characters, uppercase, lowercase, and digits).

:::info Coming Soon
Recovery phrase generation, credential management, and biometric authentication are planned for future releases.
:::

### Desktop (Admin Console)

The desktop application provides tools for network administration:

- **Network Explorer** — Interactive world map showing connected peers in the P2P network, with real-time connection status and geolocation data.
- **Daemon Control** — Start, stop, and monitor the background daemon service directly from the interface.
- **Login** — Authentication via rotating QR codes (30-second refresh cycle).
- **Multi-language** — Available in English and Spanish, auto-detected from your system language.

:::info Coming Soon
Dashboard overview, settings management, and credential issuance workflows are planned for future releases.
:::
