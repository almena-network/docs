# Authentication Overview

Learn how to authenticate your API requests to Almena ID.

## Authentication Methods

### Coming Soon

API authentication methods will be documented when the authentication system is implemented.

Planned authentication methods:
- API Keys
- OAuth 2.0
- JWT tokens

## Current Status

Currently, the Almena ID API has the following endpoints that don't require authentication:
- `GET /health` - Health check
- `GET /` - API information

All future endpoints that handle sensitive operations will require authentication.
