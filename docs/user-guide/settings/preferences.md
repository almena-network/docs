# Preferences

Customize your Almena ID experience.

## Settings Overview

Access settings by clicking **Settings** in the sidebar menu.

## Available Settings

### Language

Change the interface language:

- English
- Spanish
- French (wallet: English and Spanish only)
- German (wallet: English and Spanish only)
- Italian (wallet: English and Spanish only)

**How to change**:
1. Go to Settings
2. Select the Language dropdown
3. Choose your preferred language
4. The interface updates immediately

[Learn more about Language Settings →](./language.md)

### Biometric Authentication

Enable or disable biometric unlock for faster access to your wallet.

**How to enable**:
1. Go to Settings
2. Find the "Biometric Authentication" toggle
3. Turn it ON
4. Authenticate with your biometric sensor when prompted
5. If successful, biometric unlock is enabled

**How to disable**:
- Simply toggle the switch OFF

If your device does not support biometric authentication, the option will show as "Unavailable".

**Supported platforms**:
- **macOS**: Touch ID
- **iOS**: Touch ID / Face ID (coming soon)
- **Android**: Fingerprint / Face Unlock (coming soon)
- **Windows**: Windows Hello (coming soon)
- **Linux**: Not supported

### Auto-Lock

The wallet automatically locks after **5 minutes of inactivity**. This setting is currently fixed and cannot be changed. The information is displayed in Settings for your reference.

## Settings Data Storage

All settings are stored:
- Locally on your device
- Never sent to servers
- Specific to each device

## Troubleshooting Settings

### Settings Not Saving

**Solutions**:
1. Restart the application
2. Check device storage space
3. Try again

### Can't Access Settings

**Solutions**:
1. Ensure you're logged in
2. Restart the application
3. Recover identity if needed
