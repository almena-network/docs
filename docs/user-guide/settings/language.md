# Language Settings

Almena ID supports multiple languages. Learn how to change your interface language.

## Supported Languages

Almena ID is available in:
- 🇬🇧 **English** (en)
- 🇪🇸 **Spanish** (es) - Español
- 🇫🇷 **French** (fr) - Français
- 🇩🇪 **German** (de) - Deutsch
- 🇮🇹 **Italian** (it) - Italiano

## Changing Your Language

### Step 1: Open Settings

From your dashboard:
1. Click the **Settings** icon or button
2. You'll see the Settings screen

### Step 2: Select Language

1. Find the **Language** option
2. Click on the language dropdown
3. Select your preferred language from the list

### Step 3: Language Updates Immediately

The interface will update instantly to your chosen language.

## Language Persistence

Your language choice is:
- ✅ Saved automatically
- ✅ Persists across sessions
- ✅ Remembered after logout
- ✅ Synced when you recover your account on another device (stored locally)

## Translating Almena ID

All text in Almena ID is translated, including:
- Buttons and labels
- Instructions and help text
- Error messages
- Settings options
- Dashboard information

### Not Translated

The following remain in their original form:
- Your DID (technical identifier)
- Technical terms (DID, BIP39, etc.)
- Error codes
- Links to external resources

## Troubleshooting

### Language Not Changing

If the language doesn't update:
1. Refresh the extension (close and reopen side panel)
2. Restart your browser
3. Try selecting the language again
4. Check browser console for errors

### Missing Translation

If you see English text in other languages:
- This is likely a new feature not yet translated
- Report it to support@almena.id

## Contributing Translations

Want to help translate Almena ID to your language or improve existing translations?

Contact us:
- Email: translations@almena.id

## Language and Privacy

Your language preference:
- ✅ Is stored locally in your browser
- ✅ Is not sent to any server
- ✅ Does not identify you personally
