# Password Best Practices

Your password is the first line of defense for your Almena ID identity. Learn how to create and manage a strong password.

## Creating a Strong Password

### Minimum Requirements

Almena ID requires passwords to:
- Be at least 8 characters long
- Contain letters (uppercase and lowercase)
- Include numbers
- Have special characters (!@#$%^&*)

### Recommended Guidelines

For maximum security, your password should:
- **Be 12+ characters** (longer is better)
- **Use random combinations** (not dictionary words)
- **Mix character types** (letters, numbers, symbols)
- **Be unique** (not used elsewhere)

### Examples

❌ **Weak Passwords**:
- `password123`
- `almena2024`
- `MyName123`
- Your birthday or name

✅ **Strong Passwords**:
- `K9#mP$wR@2xL5nQ`
- `Tr!angle$Moon42#`
- `pL@y7&BooK#9Zx`

## Password Management

### Use a Password Manager

We highly recommend using a password manager like:
- 1Password
- Bitwarden
- LastPass
- Dashlane

**Benefits**:
- Generate strong random passwords
- Store passwords securely
- Auto-fill when needed
- One master password to remember

### Don't Reuse Passwords

❌ **Never** use your Almena ID password for:
- Other websites
- Email accounts
- Social media
- Banking
- Any other service

✅ **Each service should have a unique password**

## What Your Password Protects

Your Almena ID password encrypts:
- Your private key (stored locally)
- Access to your wallet
- Ability to use your identity

**Your password does NOT**:
- Get stored anywhere
- Get transmitted to servers
- Get backed up (it only exists in your memory)

## Changing Your Password

Currently, to change your password:

1. Use "Recover Account" from the login screen
2. Enter your 12-word recovery phrase
3. Create a new password
4. Your identity is restored with the new password

**Note**: Changing your password doesn't change your DID or keys, only the encryption.

## If You Forget Your Password

**Without your recovery phrase**:
- Your identity **cannot be recovered**
- This is by design for maximum security
- You'll need to create a new identity

**With your recovery phrase**:
- You can recover your identity
- See [Recovering Your Identity](../wallet/recovering-identity.md)

## Security Tips

### ✅ Do

- Use a unique, strong password
- Use a password manager
- Change it if you suspect compromise
- Keep it private

### ❌ Don't

- Share your password with anyone (including "support")
- Write it down on paper
- Store it in plain text files
- Use the same password elsewhere
- Use simple or common passwords

## Additional Security Layers

While your password is important, also secure:

1. **Recovery Phrase**: Your ultimate backup
2. **Device Security**: Lock your computer/phone
3. **Browser Security**: Keep browser updated
4. **Physical Security**: Secure physical access to devices

## Password Recovery Scams

**⚠️ Warning**: Almena ID will NEVER:
- Ask for your password via email
- Offer to "recover" your password
- Request your password in support tickets
- Call you asking for credentials

If anyone asks for your password, it's a **scam**.
