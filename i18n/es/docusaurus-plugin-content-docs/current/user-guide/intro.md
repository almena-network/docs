# Guía del Usuario

Bienvenido a la Guía del Usuario de Almena ID. Esta guía completa te ayudará a entender y usar todas las funcionalidades de Almena ID.

## Lo que Aprenderás

### Primeros Pasos
- Plataformas compatibles (Windows, macOS, Linux, Android, iOS)
- Cómo instalar Almena ID en tu dispositivo
- Crear tu primera identidad
- Entender tu DID y claves

### Usando tu Wallet
- Resumen del panel de control y navegación
- Código QR de identidad para compartir
- Enviar mensajes cifrados (DIDComm V2)
- Anclaje en blockchain para verificabilidad pública
- Autenticarse con servicios externos
- Copiar y compartir tu DID

### Aplicación Web
- Iniciar sesión con tu wallet
- Panel de control web y configuración
- Monitoreo del estado de la API

### Seguridad y Privacidad
- Mejores prácticas de contraseñas
- Gestión de frase de recuperación
- Bloqueo automático y seguridad de sesión
- Autenticación biométrica
- Funciones de privacidad

### Configuración
- Preferencias de idioma
- Habilitar autenticación biométrica

### Solución de Problemas
- Problemas comunes y soluciones
- Mensajes de error explicados
- Problemas específicos de plataforma
