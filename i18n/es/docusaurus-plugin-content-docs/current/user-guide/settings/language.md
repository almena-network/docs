# Configuración de Idioma

Almena ID soporta múltiples idiomas. Aprende cómo cambiar el idioma de tu interfaz.

## Idiomas Compatibles

Almena ID está disponible en:
- 🇬🇧 **Inglés** (en)
- 🇪🇸 **Español** (es) - Español
- 🇫🇷 **Francés** (fr) - Français
- 🇩🇪 **Alemán** (de) - Deutsch
- 🇮🇹 **Italiano** (it) - Italiano

## Cambiar tu Idioma

### Paso 1: Abre Configuración

Desde tu panel de control:
1. Haz clic en el icono o botón **Configuración**
2. Verás la pantalla de Configuración

### Paso 2: Selecciona Idioma

1. Encuentra la opción **Idioma**
2. Haz clic en el menú desplegable de idioma
3. Selecciona tu idioma preferido de la lista

### Paso 3: El Idioma se Actualiza Inmediatamente

La interfaz se actualizará instantáneamente a tu idioma elegido.

## Persistencia del Idioma

Tu elección de idioma es:
- ✅ Guardada automáticamente
- ✅ Persiste entre sesiones
- ✅ Recordada después de cerrar sesión
- ✅ Sincronizada cuando recuperas tu cuenta en otro dispositivo (almacenada localmente)

## Traducción de Almena ID

Todo el texto en Almena ID está traducido, incluyendo:
- Botones y etiquetas
- Instrucciones y texto de ayuda
- Mensajes de error
- Opciones de configuración
- Información del panel de control

### No Traducido

Lo siguiente permanece en su forma original:
- Tu DID (identificador técnico)
- Términos técnicos (DID, BIP39, etc.)
- Códigos de error
- Enlaces a recursos externos

## Solución de Problemas

### El Idioma No Cambia

Si el idioma no se actualiza:
1. Cierra y vuelve a abrir Almena ID
2. Reinicia la aplicación
3. Intenta seleccionar el idioma nuevamente

### Traducción Faltante

Si ves texto en inglés en otros idiomas:
- Esto probablemente es una nueva función aún no traducida
- Repórtalo a support@almena.id

## Idioma y Privacidad

Tu preferencia de idioma:
- ✅ Se almacena localmente en tu dispositivo
- ✅ No se envía a ningún servidor
- ✅ No te identifica personalmente
