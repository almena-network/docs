# No Puedo Iniciar Sesión

Soluciones para problemas de inicio de sesión y contraseñas.

## Contraseña Incorrecta

### Síntomas
- Error "Contraseña inválida"
- No puedes acceder a tu billetera
- La contraseña no funciona

### Soluciones

1. **Verifica la Contraseña Cuidadosamente**
   - Las contraseñas distinguen entre mayúsculas y minúsculas
   - Verifica espacios extra
   - Asegúrate de que Bloq Mayús esté desactivado
   - Intenta escribir lentamente

2. **Prueba Otro Dispositivo**
   - Si configuraste Almena ID en otro dispositivo
   - Usa ese dispositivo para acceder a tu billetera
   - Luego puedes recuperar en este dispositivo

3. **Usa la Frase de Recuperación**
   - Si has perdido tu contraseña
   - Haz clic en "Recuperar Cuenta"
   - Ingresa tu frase de recuperación de 12 palabras
   - Crea una nueva contraseña

## Contraseña Olvidada

### Si Tienes tu Frase de Recuperación

✅ **¡Puedes recuperar tu cuenta!**

1. Haz clic en "Recuperar Cuenta"
2. Ingresa tu frase de recuperación de 12 palabras
3. Crea una nueva contraseña
4. Tu identidad será restaurada

### Si No Tienes tu Frase de Recuperación

❌ **No se puede recuperar la cuenta**

Desafortunadamente, sin tu frase de recuperación, no hay forma de recuperar tu cuenta. Esto es por diseño para seguridad - no tenemos acceso a tus datos.

**Qué hacer**:
- Crea una nueva identidad
- Guarda tu nueva frase de recuperación de forma segura
- Empieza de nuevo

## Problemas con la Frase de Recuperación

### Frase de Recuperación Incorrecta

**Síntomas**:
- Error "Frase de recuperación inválida"
- La frase no es aceptada
- No puedes recuperar la cuenta

**Soluciones**:

1. **Verifica el Orden de las Palabras**
   - Las palabras deben estar en orden exacto
   - Palabra 1, luego palabra 2, etc.
   - ¡El orden importa!

2. **Verifica la Ortografía**
   - Cada palabra debe estar escrita exactamente correcta
   - Usa letras minúsculas
   - Sin errores tipográficos

3. **Verifica el Conteo de Palabras**
   - Debe ser exactamente 12 palabras
   - No 11, no 13
   - Cuenta cuidadosamente

4. **Prueba Formato Diferente**
   - Algunas personas agregan números (1. palabra, 2. palabra)
   - Ingresa solo las palabras, sin números
   - Elimina cualquier puntuación

### No Puedo Encontrar la Frase de Recuperación

**Dónde buscar**:
- Gestor de contraseñas (si la guardaste allí)
- Aplicaciones de notas seguras
- Foto de copia escrita a mano
- Papel físico en ubicación segura
- Otros dispositivos donde configuraste Almena ID

**Si está verdaderamente perdida**:
- No se puede recuperar la cuenta
- Debes crear una nueva identidad
- El DID anterior está permanentemente perdido

## Cuenta Bloqueada

### Demasiados Intentos Fallidos

La aplicación puede ralentizarse después de muchos intentos fallidos de contraseña.

**Soluciones**:
1. Espera 15 minutos
2. Cierra la billetera completamente
3. Vuelve a abrir e intenta nuevamente
4. Si aún está bloqueada, reinicia el dispositivo

## Billetera No Responde

Si la billetera no responde a los intentos de inicio de sesión:

1. **Reinicia la Aplicación**
   - Cierra Almena ID completamente
   - Vuelve a abrirlo
   - Intenta nuevamente

2. **Reinicia el Dispositivo**
   - Si el problema persiste
   - Reinicia tu dispositivo
   - Intenta iniciar sesión

3. **Reinstala si es Necesario**
   - Desinstala Almena ID
   - Reinstala
   - Recupera con la frase de recuperación

## Dispositivo Diferente

### Usando un Nuevo Dispositivo

Si configuraste tu identidad en un dispositivo pero ahora usas otro:

**Nota**: Cada dispositivo almacena datos por separado. Necesitas recuperar tu identidad en el nuevo dispositivo usando tu frase de recuperación.

1. Instala Almena ID en el nuevo dispositivo
2. Haz clic en "Recuperar Cuenta"
3. Ingresa la frase de recuperación
4. Crea una contraseña para este dispositivo

## Datos No Encontrados

### Error "No se Encontró Billetera"

Esto significa que no hay identidad configurada en este dispositivo.

**Soluciones**:
- Si es la primera vez: Crea una nueva cuenta
- Si tenías una cuenta: Recupera con la frase de recuperación

## Consejos de Prevención

### Evitar Problemas de Inicio de Sesión

✅ **Haz**:
- Escribe la frase de recuperación inmediatamente
- Guarda la frase de recuperación en múltiples lugares seguros
- Usa un gestor de contraseñas
- Mantén la contraseña fuerte pero memorable
- Configura en múltiples dispositivos

❌ **No Hagas**:
- Dependas solo de la memoria
- Compartas la frase de recuperación
- Uses la misma contraseña en todas partes
- Omitas escribir la frase de recuperación

## Nota de Seguridad

🔒 **No podemos restablecer tu contraseña**

Esto es por diseño. Tus datos están encriptados con tu contraseña, y no tenemos acceso a ellos. Solo tu frase de recuperación puede restaurar el acceso.

Esto asegura que tu identidad sea verdaderamente tuya y no pueda ser accedida por nadie más, incluyéndonos a nosotros.
