# Billetera No Funciona

Soluciones para cuando la billetera Almena ID no responde o no funciona correctamente.

## Problemas Comunes

### La Billetera No Se Abre

**Síntomas**:
- Hacer clic en el icono de la aplicación no hace nada
- La ventana de la aplicación no aparece
- El icono de la aplicación está en gris o no responde

**Soluciones**:

1. **Reinicia la Aplicación**
   - Cierra la billetera completamente
   - Espera unos segundos
   - Ábrela nuevamente

2. **Reinicia tu Dispositivo**
   - Cierra todas las aplicaciones
   - Reinicia tu computadora o dispositivo móvil
   - Intenta abrir la billetera nuevamente

3. **Verifica Actualizaciones**
   - Asegúrate de tener la última versión instalada
   - Actualiza a través de tu tienda de aplicaciones o descarga la última versión

4. **Reinstala la Billetera**
   - Desinstala la aplicación
   - Reinicia tu dispositivo
   - Instala la billetera nuevamente
   - Tus datos permanecen seguros si tienes tu frase de recuperación

### La Billetera Aparece Congelada

**Síntomas**:
- La interfaz no responde a los clics
- Los botones no funcionan
- Atascado en la pantalla de carga

**Soluciones**:

1. **Fuerza Cierre y Vuelve a Abrir**
   - Fuerza el cierre de la aplicación
   - Espera unos segundos
   - Ábrela nuevamente

2. **Borra la Caché de la Aplicación** (Escritorio)
   - Cierra la billetera
   - Navega a la carpeta de datos de la aplicación
   - Elimina los archivos de caché
   - Reinicia la billetera

3. **Verifica los Recursos del Sistema**
   - Cierra otras aplicaciones para liberar memoria
   - Verifica si tu dispositivo tiene suficiente espacio de almacenamiento
   - Reinicia tu dispositivo si es necesario

### Problemas Específicos de Móvil

**Android**:
- Verifica si la optimización de batería está deshabilitando la aplicación
- Asegúrate de que la aplicación tenga los permisos necesarios
- Borra la caché de la aplicación en Configuración → Aplicaciones → Almena ID → Borrar Caché

**iOS**:
- Fuerza el cierre y vuelve a abrir la aplicación
- Verifica actualizaciones de iOS
- Reinstala si los problemas persisten
