# Mejores Prácticas de Contraseñas

Tu contraseña es la primera línea de defensa para tu identidad Almena ID. Aprende cómo crear y gestionar una contraseña fuerte.

## Crear una Contraseña Fuerte

### Requisitos Mínimos

Almena ID requiere que las contraseñas:
- Tengan al menos 8 caracteres de longitud
- Contengan letras (mayúsculas y minúsculas)
- Incluyan números
- Tengan caracteres especiales (!@#$%^&*)

### Pautas Recomendadas

Para máxima seguridad, tu contraseña debería:
- **Tener 12+ caracteres** (más largo es mejor)
- **Usar combinaciones aleatorias** (no palabras del diccionario)
- **Mezclar tipos de caracteres** (letras, números, símbolos)
- **Ser única** (no usada en otro lugar)

### Ejemplos

❌ **Contraseñas Débiles**:
- `password123`
- `almena2024`
- `MiNombre123`
- Tu cumpleaños o nombre

✅ **Contraseñas Fuertes**:
- `K9#mP$wR@2xL5nQ`
- `Tr!angle$Moon42#`
- `pL@y7&BooK#9Zx`

## Gestión de Contraseñas

### Usa un Gestor de Contraseñas

Recomendamos encarecidamente usar un gestor de contraseñas como:
- 1Password
- Bitwarden
- LastPass
- Dashlane

**Beneficios**:
- Genera contraseñas aleatorias fuertes
- Almacena contraseñas de forma segura
- Auto-completa cuando es necesario
- Una contraseña maestra para recordar

### No Reutilices Contraseñas

❌ **Nunca** uses tu contraseña de Almena ID para:
- Otros sitios web
- Cuentas de correo electrónico
- Redes sociales
- Banca
- Cualquier otro servicio

✅ **Cada servicio debe tener una contraseña única**

## Qué Protege tu Contraseña

Tu contraseña de Almena ID encripta:
- Tu clave privada (almacenada localmente)
- Acceso a tu billetera
- Capacidad de usar tu identidad

**Tu contraseña NO**:
- Se almacena en ningún lugar
- Se transmite a servidores
- Se respalda (solo existe en tu memoria)

## Cambiar tu Contraseña

Actualmente, para cambiar tu contraseña:

1. Usa "Recuperar Cuenta" desde la pantalla de inicio de sesión
2. Ingresa tu frase de recuperación de 12 palabras
3. Crea una nueva contraseña
4. Tu identidad se restaura con la nueva contraseña

**Nota**: Cambiar tu contraseña no cambia tu DID o claves, solo la encriptación.

## Si Olvidas tu Contraseña

**Sin tu frase de recuperación**:
- Tu identidad **no puede ser recuperada**
- Esto es por diseño para máxima seguridad
- Necesitarás crear una nueva identidad

**Con tu frase de recuperación**:
- Puedes recuperar tu identidad
- Consulta [Recuperar tu Identidad](../wallet/recovering-identity.md)

## Consejos de Seguridad

### ✅ Haz

- Usa una contraseña única y fuerte
- Usa un gestor de contraseñas
- Cámbiala si sospechas compromiso
- Manténla privada

### ❌ No Hagas

- Compartas tu contraseña con nadie (incluyendo "soporte")
- La escribas en papel
- La almacenes en archivos de texto plano
- Uses la misma contraseña en otro lugar
- Uses contraseñas simples o comunes

## Capas Adicionales de Seguridad

Mientras tu contraseña es importante, también asegura:

1. **Frase de Recuperación**: Tu respaldo definitivo
2. **Seguridad del Dispositivo**: Bloquea tu computadora/teléfono
3. **Seguridad del Navegador**: Mantén el navegador actualizado
4. **Seguridad Física**: Asegura el acceso físico a los dispositivos

## Estafas de Recuperación de Contraseñas

**⚠️ Advertencia**: Almena ID NUNCA:
- Te pedirá tu contraseña por correo electrónico
- Ofrecerá "recuperar" tu contraseña
- Solicitará tu contraseña en tickets de soporte
- Te llamará pidiendo credenciales

Si alguien te pide tu contraseña, es una **estafa**.
