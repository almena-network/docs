# Resumen de Autenticación

Aprende cómo autenticar tus solicitudes a la API de Almena ID.

## Métodos de Autenticación

### Próximamente

Los métodos de autenticación de la API se documentarán cuando el sistema de autenticación esté implementado.

Métodos de autenticación planeados:
- API Keys
- OAuth 2.0
- Tokens JWT

## Estado Actual

Actualmente, la API de Almena ID tiene los siguientes endpoints que no requieren autenticación:
- `GET /health` - Verificación de estado
- `GET /` - Información de la API

Todos los endpoints futuros que manejen operaciones sensibles requerirán autenticación.

## Mantente Actualizado

Consulta el [Registro de Cambios](../../changelog/overview.md) para actualizaciones sobre la implementación de autenticación.
